## Project Overview
This is a text-based Java application developed for ICT711 Programming and Algorithms.
The system manages aged care residents by allowing an admin to add, search, update, delete, and evaluate resident records.

## Features
- Add resident
- Search resident by ID or name
- Update resident details
- Delete resident
- Evaluate resident health status
- Read data from a file
- Save data to file

## Concepts Used
- Classes and objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- ArrayList
- File handling
- Exception handling

## How to Run
1. Open the project in Eclipse
2. Run Main.java
3. Follow the menu in the console
